# 学校管理系统 python301

## 项目概述

学校管理系统是一款旨在提高学校管理工作效率的软件，采用 Python 语言结合 Flask 框架，以及 MySQL 数据库构建而成。

## 技术栈

- **后端语言**: Python
- **Web 框架**: Flask
- **数据库**: MySQL

## 功能模块

- 学生信息管理
- 教师信息管理
- 课程信息管理
- 成绩管理
- 用户权限控制

## 系统角色

- 管理员
- 教师
- 学生

## 运行环境

- Python 3.x
- Flask 1.x
- MySQL 5.x

## 部署步骤

1. 安装 Python 环境及依赖库
2. 安装配置 MySQL 数据库
3. 部署应用代码
4. 执行数据库初始化脚本
5. 启动后端服务

## 目录结构

```
school_management_system/
├── app/                    # Flask 应用目录
│   ├── models/             # 数据模型目录
│   ├── views/              # 视图层目录
│   ├── templates/          # HTML 模板目录
│   ├── static/             # 静态文件目录
│   ├── __init__.py         # Flask 应用初始化文件
│   └── ...
├── venv/                   # Python 虚拟环境目录
├── requirements.txt        # 依赖列表
├── setup.py                # 安装脚本
└── ...
```
